# App Store Screenshots

Place the final iPhone App Store screenshots for Tennis Agent in this folder so the full submission package lives in GitHub.

Recommended filenames:

- `01-landing-screen.png`
- `02-main-menu.png`
- `03-senior-players.png`
- `04-tournament-selection.png`
- `05-results-or-player-details.png`

Notes:

- Capture real in-app screenshots from an iPhone simulator or device.
- Keep status bars clean and use the same device size for the full set.
- Upload the same PNGs you store here to App Store Connect.
